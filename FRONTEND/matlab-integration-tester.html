🔧 Troubleshooting

Common Issues:
  • MATLAB not found: Make sure MATLAB is in your system PATH
  • Permission errors: Check if Node.js can create temp files
  • Audio format issues: Ensure audio is in WAV format
  • Timeout errors: Increase timeout if analysis takes longer

Fix MATLAB Path (if needed):

If MATLAB is not in PATH, update the command in server.js:

// Replace this line in server.js:
const matlabCommand = `matlab -batch "addpath('${__dirname}'); cough_analysis('${tempFilePath}')"`;

// With full MATLAB path:
const matlabCommand = `"C:\\Program Files\\MATLAB\\R2023b\\bin\\matlab.exe" -batch "addpath('${__dirname}'); cough_analysis('${tempFilePath}')"`;
