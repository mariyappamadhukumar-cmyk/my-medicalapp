## 🎯 How It Works

### Step 1: User Describes Symptoms
```
User: "I have fever, headache, and body pain"
```

### Step 2: AI Analyzes & Diagnoses
```
🏥 Diagnosis: Viral Flu (Influenza)
Explanation: Common viral infection causing fever, body aches, and fatigue
```

### Step 3: AI Suggests Medications
```
💊 Recommended Medications:

📋 Tablets/Oral Medicines:
• Paracetamol - 500mg, every 6 hours, for 3-5 days
  Brand names: Tylenol, Panadol
  ⚠️ Do not exceed 4000mg per day

• Vitamin C - 1000mg, once daily, for 7 days
  Supports immune system

• Cetirizine - 10mg, once daily, for runny nose
  Brand names: Zyrtec
```

### Step 4: Suggests Vaccines
```
💉 Recommended Vaccines:
• Annual Flu Vaccine - Prevents influenza infections
• COVID-19 Booster - If not taken in last 6 months
```

### Step 5: Provides Home Care
```
🏡 Home Remedies:
• Rest and stay well-hydrated
• Drink warm liquids (tea, soup)
• Steam inhalation for congestion
• Maintain good nutrition
```

### Step 6: Safety Guidance
```
🚨 When to See a Doctor:
• Fever persisting more than 3 days
• Difficulty breathing
• Severe headache or stiff neck
• Confusion or altered consciousness
```
