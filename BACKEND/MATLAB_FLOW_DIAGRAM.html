[~, idx] = max(abs(Y)) ← Find dominant frequency

📊 Feature Extraction

MATLAB calculates spectral features

spectralCentroid = sum(f .* P) / sum(P)
spectralBandwidth = sqrt(sum((f - centroid)^2 .* P) / sum(P))
zeroCrossingRate = sum(abs(diff(sign(audio)))) / length(audio)
rmsEnergy = sqrt(mean(audio.^2))

🔧 MATLAB Code Example

function result = cough_analysis(audioFilePath) % Load audio file [audioData, fs] = audioread(audioFilePath); % Convert to mono if size(audioData, 2) > 1 audioData = mean(audioData, 2); end % FFT Analysis N = length(audioData); Y = fft(audioData); P1 = abs(Y/N); f = fs*(0:N/2)/N; % Find dominant frequency [maxAmplitude, maxIdx] = max(P1); dominantFrequency = f(maxIdx); % Classify cough type if dominantFrequency < 250 pattern = 'Very low-frequency wet cough'; elseif dominantFrequency < 400 pattern = 'Low-frequency wet cough'; elseif dominantFrequency < 600 pattern = 'Moderate dry cough'; else pattern = 'High-frequency dry cough'; end % Save results result = struct('dominantFrequency', dominantFrequency, ... 'pattern', pattern, ... 'confidence', 70); end
